grid-path-finder NPM Package

An NPM package that finds the shortest path between two points in a grid maze. The approach is naive: recursively checking each direction and backtracking at dead-ends. Each node records its distance from the start to check if a shorter path already exists. I hope to rewrite this in a non-recursive manner, before attempting a more sophisticated approach such as Dijkstra's algorithm. Previews pending.
